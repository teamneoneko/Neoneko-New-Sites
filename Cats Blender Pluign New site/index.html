Cats Blender Plugin

A tool designed to shorten steps needed to import and optimize models.

Cats Blender Plugin helps you import and optimize 3D models for VRChat and other platforms. It supports MMD, XNALara, Mixamo, DAZ/Poser, Blender Rigify, Sims 2, Motion Builder, 3DS Max and potentially more model types.
